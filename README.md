# Hospital Resource Utilization Dashboard

## Project Overview
This project analyzes hospital resource utilization and inventory management using Power BI and Python.

## Tools Used
- Power BI Desktop
- Python
- Google Colab
- Pandas

## Files
- HospitalDashboard.pbix
- Hospital_Analysis.ipynb
- inventory_data.xlsx
- hospital_bed_resource_management.xlsx

## Features
- KPI Cards
- Interactive Charts
- Tables
- Data Analysis using Python

## Key Conclusions

1. ICU beds have the highest occupancy rate.
2. Hospital A handles the largest number of patients.
3. Bed availability varies significantly between hospitals.
4. Emergency admissions increase bed utilization.
5. Better resource allocation could improve bed availability.
